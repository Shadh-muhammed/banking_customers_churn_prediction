# 📊 Banking Customer Churn Prediction Using Machine Learning

## 🚀 Project Overview
This is an end-to-end Machine Learning project that predicts customer churn using bank data. The goal is to help businesses identify customers who are likely to leave.

---

## 🧠 Key Learnings
- Data preprocessing and cleaning
- Exploratory Data Analysis (EDA)
- Feature engineering
- Model building and evaluation

---

## 📂 Dataset
Features include:
- Credit Score
- Geography
- Gender
- Age
- Balance
- Tenure
- Number of Products

---

## ⚙️ Tech Stack
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- XGBoost

---

## 🔍 Workflow

### 1. Data Preprocessing
- Handling missing values
- Encoding categorical variables
- Feature scaling

### 2. EDA
- Distribution plots
- Correlation heatmap

### 3. Model Building
- Logistic Regression
- Random Forest
- XGBoost

### 4. Evaluation
- Accuracy
- Precision
- Recall
- F1 Score
- ROC-AUC

---

## 📈 Results
- Tree-based models performed better
- Key churn factors identified:
  - Age
  - Balance
  - Activity level

---

## 💼 Business Impact
- Helps in customer retention
- Reduces churn rate
- Improves decision making
